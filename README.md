# 크롤링 개발
Node.js 기반 크롤링 모듈 개발


# 실행방법
1. server.js 실행 ⇒ node ./config/server.js

 5000 번 포트 백앤드 서버 실행

2. react 실행 ⇒ yarn start

 3000 번 포트 프론트 앤드 서버 실행
 
# 동작 원리

백앤드 서버로 요청이 이루어지면, 크롤링한 데이터를 가져와 DB연결 없이 브라우저로 데이터를 전송하여, 랜더링이 이루어질때마다 실시간 정보를 받아올 수 있도록 구현하였습니다.
